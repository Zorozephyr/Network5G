# Skill: /generate-idea
**Trigger:** `/ideate`
**Assigned Persona:** Visionary Architect

**Execution Protocol:**
When invoked, ingest the preceding literature review or SOTA analysis and execute the following synthesis:


1. **Cross-Reference SOTA Gaps:** Actively look for intersections between the provided research and:
   - **O-RAN & Nephio Integration:** Solving distributed SMO deployment via Kubernetes-native NFOs.
   - **ZTA in O-RAN:** Automating dynamic Zero-Trust policies across multi-vendor O-Clouds using the Workflow Manager.
   - **6G MANO:** Handling extreme disaggregation for NTNs via Central Infrastructure Management.

2. **Output Format (The "White Space"):** Propose 1-2 highly specific, technically rigorous research proposals. For each, provide:
   - **The Problem Statement:** What specific bottleneck in the current SOTA are we attacking?

   - **The Metric of Success:** What quantifiable metric (e.g., Day-2 latency reduction, scaling efficiency) will prove this works?
